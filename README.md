# カブスコープ - 株チェックAI

日本株の初心者向けポートフォリオ管理アプリ。ブラウザだけで動作します（サーバー不要）。

## 機能

- **ダッシュボード** — 保有銘柄の一覧・評価損益・集中度チェック
- **リアルタイム株価** — Yahoo Finance から現在値・前日比を自動取得（5分キャッシュ）
- **スクショで追加** — 証券会社アプリのスクリーンショットから銘柄を自動読み取り（Tesseract.js OCR）
- **手動で追加** — 約200銘柄＋主要ETFのマスターデータ。マスターにない銘柄も4桁コード入力で追加可能（名称・株価は自動取得）
- **AI診断** — 集中リスク・セクター偏り・分散状況を自動判定してA〜Dでスコアリング
- **推奨銘柄** — ポートフォリオに足りないセクターの安定銘柄を提案
- **資産推移** — 日次スナップショットの自動記録とグラフ表示
- **配当シミュレーター** — 年間配当見込み・銘柄別内訳・10年予測。厳選銘柄に加え、全上場銘柄の配当も直近実績（`dividends.json`）から自動反映
- **AI相談** — Claude API連携の本物のAIチャット（APIキー設定時）。未設定時は定型応答

## 使い方

[GitHub Pages で公開中のページ](../../deployments) を開くだけ。またはリポジトリをcloneして `index.html` をブラウザで開いてください。

### AI相談（Claude API）の設定

1. [platform.claude.com](https://platform.claude.com/) で Anthropic APIキーを取得
2. アプリの「AI相談」ページ →「AI設定」からキーを入力

APIキーは**お使いのブラウザの localStorage にのみ**保存されます。このリポジトリやサーバーに送信されることはありません（通信先は Anthropic のAPIのみ）。

## データについて

- ポートフォリオデータはすべてブラウザの localStorage に保存されます（外部送信なし）
- 株価は Yahoo Finance の非公式APIを利用した参考値です
- 配当額・利回りは、厳選銘柄は目安値、その他の銘柄は直近12ヶ月の配当実績（Yahoo Finance）からの概算です。いずれも将来の配当を保証しません。投資判断の前に必ず証券会社等で最新情報をご確認ください
- 本アプリは投資助言を行うものではありません。投資は自己責任でお願いします

## 技術構成

- Vanilla JavaScript / HTML / CSS（ビルド不要・依存なし）
- OCR: [Tesseract.js](https://tesseract.projectnaptha.com/)（CDN）
- 株価: Yahoo Finance chart API
- AIチャット: [Claude API](https://platform.claude.com/)（claude-opus-4-8）
